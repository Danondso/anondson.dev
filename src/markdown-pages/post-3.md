A web app music player backed by Spotify's API, written using handlebar templates and jQuery for some of the smaller bits. This project is still under construction and isn't currently deployed anywhere.

---

### Tech Stack

- [HandleBars](https://handlebarsjs.com/)
- [ExpressJS](https://expressjs.com/)
- [Spotify API](https://developer.spotify.com/documentation/web-api/)
- [jQuery](https://jquery.com/)
